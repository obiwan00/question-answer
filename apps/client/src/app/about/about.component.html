🇺🇸 About "Question Answer" project

This is a pet-project for educational purpose. It is a simplified clone of StackOverflow with chats for each topic. Rest Api


🇺🇦 Про "Question Answer" проект

Це навчальний проект, спрощена версія StackOverflow з чатами для кожного обговорення. Rest Api


🇺🇸 Technologies used / 🇺🇦 Використані технології:

  • Front end: Angular 14, Angular Material 14, Socket.io(WebSocket for chats);
  • Back end: NestJS 9, Typeorm, Socket.io(WebSocket for chats);
  • Monorepo: nrwl/nx;
  • DataBase: PostgreSQL.
